# UV Editor Extracted

Эта папка создана как отдельная вынесенная копия кода UV-редактора из текущего проекта.

Важно:

- рабочая логика проекта не менялась;
- существующие файлы UV-редактора не редактировались;
- сюда положены точные копии основных UV-файлов и отдельные пояснения по архитектуре.

## Что скопировано без изменений

- `src/js/UVEditor.js`
- `src/views/uv_editor.pug`
- `src/styles/_uv_editor.scss`

## Что не копировалось, но используется UV-редактором

UV-редактор в оригинальном проекте опирается на общие зависимости и точки интеграции:

- `src/js/ResizeableWindow.js` — базовая логика перемещаемых и ресайзимых окон
- `src/js/I18n.js` — все `panel.uv` и `uv.*` строки интерфейса
- `src/js/VSCodeContext.js` — мост состояния и сообщений с VS Code webview
- `three` и `GLTFExporter` — текстуры, канвас-текстуры и экспорт модели
- `src/js/Panel.js` — создание и показ UV-редактора
- `src/js/UIController.js` — прокидывание выбора объекта в UV-редактор
- `src/views/panel.pug` — кнопка панели и include шаблона UV-редактора
- `src/styles/style.scss` — подключение `_uv_editor.scss`
- `extension.js` — сохранение состояния UV-редактора, экспорт PNG и сохранение GLB

## Файлы с объяснением

- `LOGIC.md` — внутренняя логика и жизненный цикл UV-редактора
- `INTEGRATION.md` — где именно UV-редактор подключён в остальном проекте

## Цель этой папки

Эта папка нужна как отдельный читаемый срез UV-редактора:

- чтобы быстро смотреть только UV-код;
- чтобы не искать UV-логику по всему проекту;
- чтобы иметь рядом описание, не затрагивая рабочую реализацию.
